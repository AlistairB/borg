#  A Haskell Toy Robot

Using:

- Lenses
- State
- Writer
- Parser Combinators
- Property Based Testing

## Requires

[Haskell Stack](https://www.haskellstack.org)

## Running

### Run a REPL

```bash
auto/run
```

### Run a file

```bash
auto/run data/ex1.txt
```

## Testing

```bash
auto/test
```
